# Corevia Systems Pvt Ltd Website

A modern, premium static company website built with **React + Vite + Tailwind CSS** for **Corevia Systems Pvt Ltd**.

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Lucide React Icons

## Features

- Fully responsive landing page (mobile + desktop)
- Hero, About, Services, Tech Stack, Projects, Contact, Footer sections
- SEO-ready metadata in `index.html`
- Dark mode toggle with localStorage persistence
- Floating WhatsApp CTA
- Clean, fast-loading UI optimized for static deployment

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for production

```bash
npm run build
npm run preview
```

## Deploy on Vercel

1. Push this project to GitHub.
2. Import the repo in [vercel.com](https://vercel.com).
3. Framework preset: **Vite**.
4. Build command: `npm run build`
5. Output directory: `dist`
6. Click deploy.

Domain can then be connected to `coreviasystems.io` in Vercel project settings.
